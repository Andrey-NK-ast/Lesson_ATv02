# Ипотечный калькулятор 

Веб-приложение для расчёта параметров ипотеки и поддержкой досрочного погашения.

## Возможности

- Ввод суммы кредита, процентной ставки, срока (в годах или месяцах), первоначального взноса
- Учёт досрочного погашения: разово, ежемесячно, раз в 2 месяца, квартал, полгода, год
- Два варианта досрочной логики: уменьшить срок или платёж
- Визуализация: диаграмма (основной долг/проценты), молнии, огонь, график платежей

## Запуск

1. Установите зависимости:
```Терминале
pip install -r requirements.txt
```

2. Запустите приложение:
```Терминале
python main.py
```

3. Откройте в браузере:
```
http://127.0.0.1:5000
```

## Использование

- Введите параметры
- (Выберите опции досрочного погашения при необходимости)
- Нажмите "Рассчитать"
- Получите платеж, подробнейший график и диаграмму!
